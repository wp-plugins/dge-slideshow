=== DGE_Slideshow ===
Tags: slideshow, zooomr, flickr, xsl, xslt, javascript
Contributors: delcock

Turns online images (e.g. Flickr or Zooomr image feeds) into a slideshow in a Wordpress page or post. It's fairly flexible, due to use of XSLT.

See the <a href="http://dev.wp-plugins.org/wiki/dge-slideshow">wiki page\
</a> for detailed information, and the <a href="http://dave.stufftoread.net/slideshow/">home page</a> for examples.

== Installation ==

1. Download and install, and activate <a href="http://dev.wp-plugins.org/wiki/dge-inlinerss">DGE_InlineRSS</a> 0.92 or later first.
2. Upload this plugin to your plugins folder, usually `wp-content/plugins/`. It must be extracted to a folder called dge-slideshow.
3. Activate the plugin on the plugin screen.
4. Configure any preferences in the Options>>Slideshow menu in Wordpress admin.
5. Consult the <a href="http://dev.wp-plugins.org/wiki/dge-slideshow">wiki page</a> for usage instructions.
